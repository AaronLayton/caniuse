caniuse api

Powered by the services powering caniuse.com.

How to use

url: [ features ].[ format ][ ? [ actions ] ]

features

The features (+ delimited) to be tested, like apng or canvas+json.

format

The format the output will be returned as, like json or xml.

actions

The actions to be applied to the data, like callback=myFunction or fail=1.

Examples

canvas+json.js?callback=myFunction: detect browsers supporting both the 'canvas' and 'json' features and output the results as JavaScript wrapped it in a 'myFunction' callback.

apng.html?fail=1: detect browsers supporting 'apng' feature and output the results as HTML as though the client browser had failed the test.

Actions

callback=[ functionName ]: the JavaScript output will be wrapped in this function name.

size=[ large | small | regular ]: the HTML output will use this size for the browser icons.

text: the HTML output will include text about the browsers.

icon: the HTML output will include the browser icons.

texticon: the HTML output will include text about browsers and icons.
